Verbe Transitif Direct (及物動詞)

Renverser

本身是直接及物動詞，受詞前不需要加介系詞。但如果要表達「打翻在某個地方」或「把 A 倒進 B」，則需要搭配空間介係詞。

結構一 🛑 表面 (On)

Renverser + Qch sur + 地點

當液體或物體潑灑在某個表面上時，大腦會將 sur 視為空間的終點。

Il a renversé son café sur son clavier.

他把咖啡打翻在鍵盤上。

結構二 📦 內部 (In)

Renverser + Qch dans + 容器

當物體是跌入或傾倒進一個封閉空間或範圍時，使用 dans。

Le vent a renversé la clôture dans le jardin.

風把圍欄吹倒在花園裡。

結構三 🔄 反身 (Reflexive)

Se renverser dans / sur...

代動詞用法（自己倒下/翻車）。當主詞自己翻覆時使用，後面常搭配 dans 或 sur。

La voiture s'est renversée dans le fossé.

汽車翻覆在水溝裡。

⚡ 隨堂小測驗 (Quiz)

根據情境，選擇最適合的介系詞填空：

Q1. J'ai renversé de l'eau ______ mon livre de français. (我把水打翻在我的法文課本上。)
